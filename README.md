# Rahul Sharma Portfolio

Personal portfolio built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Development

```sh
pnpm install
pnpm dev
```

## Build

```sh
pnpm build
pnpm preview
```

## Deploy to GitHub Pages

This repository includes a Pages workflow at `.github/workflows/deploy.yml`.

1. Push your code to GitHub.
2. Open `Settings -> Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to your configured deploy branch.

After a successful workflow run, your site will be available at:
`https://<your-username>.github.io/<your-repo>/`
